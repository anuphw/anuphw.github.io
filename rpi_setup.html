Raspberry Pi Setup

0. Buy Raspberry Pi

You need following:

  • Raspberry Pi 3 Model B
  • HDMI cable
  • Monitor with HDMI input
  • USB Keyboard/mouse
  • Micro SD card (64 GB) U1 or U3

1. Installing Raspbian on your PI

2. Installing required softwares

Once the system is installed, reboot and login using following credentials:

username: pi
password: raspberry

2.1. Turning on WIFI.

You can skip this step if you are planning to use wired connection. Follow the instructions in the following video to connect to wifi.

2.1. Open terminal located in menu bar

The following command will download download a script that will set up the server on Raspberry Pi. You will need internet connection on the raspberry pi. (You can set up wifi by clicking on wifi button in the upper right corner on the screen).

wget "https://anuphw.github.io/script.sh" -O /home/pi/script.sh

Once you are done with downloading the script run following command. O

sudo bash /home/pi/script.sh

That's it! Once you are able to execute the last command without errors, it will output you an IP address. Now you can visit "http://the.ip.address/VideoCoding/test.php" (replace the.ip.address with the ip address you get).

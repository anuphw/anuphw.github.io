Raspberry Pi Setup

0. Buy Raspberry Pi

You need following:

  • Raspberry Pi 3 Model B
  • HDMI cable
  • Monitor with HDMI input
  • USB Keyboard/mouse
  • Micro SD card (64 GB) U1 or U3
  • Ethernet Cable (if you don't want to use wifi)

1. Download NOOBS:

Download NOOBS OS from https://www.raspberrypi.org/downloads/noobs/. Download the Offline and network install version.

2. Prepare OS

Now attach the SD card to the computer and format it with FAT32 file system. Unzip the NOOBS OS (Operating System) in a folder and copy the contents of that folder (not the folder) to the formatted SD card.

3. Preparing the PI [:)]

3.1 SD Card

Safely remove the SD card and put it into the PI.

3.2 Attach Cables and Power on.

Attach keyboard, mouse, HDMI and Ethernet (if you're not using wifi) after inserting the SD card into PI. Once this is done, you can connect the PI to a power source. You should be connected to internet using a wired or wireless connection (either of them is fine).

3.3 Installing Rasbian.

Once you attach the power supply the PI will boot up and you can start installing the OS.

At the end of installation process, you will get a message saying OS(es) Installed Sucessfully. Click OK and the PI will reboot.

4. Installing required softwares.

Once the system is installed, reboot and login using following credentials:

username: pi
password: raspberry

After the reboot you will need to install apache2 and php. You need to be connected to the internet for the following process.

4.1. Turning on WIFI.

You can skip this step if you are planning to use wired connection. Follow the instructions in the following video to connect to wifi.

4.2. Open terminal located in menu bar
4.3. Open a browser

Open a browser and open these instructions (https://anuphw.github.io/rpi_setup.html) on PI and continue.

4.4. Download and install

Copy the following line and paste into the terminal window and hit ENTER.

wget "https://anuphw.github.io/script.sh" -O /home/pi/script.sh

Once the script is downloaded, copy the following line and paste it into the terminal window then hit ENTER. In case if you need to enter password, the password is raspberry. This will install the required files for coding and it will also download videos for M. Ozil.

sudo bash /home/pi/script.sh

In case you are asked for yes/no, just say yes.

5. Try the server.

At the end of the installation, the script will also tell you the IP address of the PI. Now you can visit "http://the.ip.address/VideoCoding/test.php" (replace the.ip.address with the ip address you get). For example if your ip address is 10.10.1.100, open a browser in any computer that is connected to same network/wifi as raspberry pi, and enter "http://10.10.1.100/VideoCoding/test.php"
